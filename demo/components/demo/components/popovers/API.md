# `<ngl-popover #ref>`

# `[nglPopoverTrigger]`

 ### Input

   * `nglPopoverTrigger: NglPopover`: The connected popover reference to show.
   * `nglOpen: boolean = false`: Whether popover is visible.
   * `nglPlacement: 'top' | 'right' | 'bottom' | 'left' = 'top'`: Position relative to host element.
